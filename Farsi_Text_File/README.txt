# FarsiWordList4Games

یک لیست تمیز و قابل استفاده از کلمات فارسی برای استفاده در بازی‌ها و پروژه‌های گیم‌دولوپمنت. این پروژه با هدف فراهم کردن یک منبع عمومی از کلمات فارسی طراحی شده که:

- برای استفاده در بازی‌های کلمه‌ای و زبان‌محور ساده‌سازی شده است.
- از حروف و علامت‌های دشوار مانند «ء» (همزه) یا «أ» و «إ» که در گیم‌پلی و تایپ دشواری ایجاد می‌کنند، پاک‌سازی شده است.
- به صورت مداوم به‌روزرسانی می‌شود.
- امکان مشارکت و توسعه‌ی آن توسط دیگران فراهم است.

## اهداف پروژه

- ساخت یک فایل متن سبک و کاربردی برای بازی‌های فارسی‌زبان.
- کمک به بازی‌سازان برای پیاده‌سازی سریع بازی‌های واژه‌ای، حدس کلمه و ... .
- حذف یا ساده‌سازی حروف و نشانه‌هایی که در رابط کاربری بازی قابل نمایش یا تایپ نیستند.

## فرمت فایل

فایل اصلی پروژه به صورت متنی (`.txt`)آماده می‌شود تا در همه‌ی موتورهای بازی‌سازی (مثل Unity، Godot یا Unreal) قابل استفاده باشد.

## مشارکت

از همه‌ی علاقه‌مندان دعوت می‌شود که با ارسال کلمات جدید، بهبود لیست فعلی، یا ساخت ابزارهایی برای بررسی صحت واژه‌ها به پروژه کمک کنند.

## لایسنس

این پروژه تحت مجوز MIT منتشر شده و استفاده از آن برای همه آزاد است.
