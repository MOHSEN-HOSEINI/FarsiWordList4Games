# راهنمای مشارکت در پروژه FarsiGameWords

از مشارکت شما در این پروژه سپاسگزاریم! لطفاً پیش از ارسال Pull Request موارد زیر را رعایت فرمایید:

## قوانین محتوا

- **از اضافه کردن کلمات توهین‌آمیز، فحاشی، الفاظ رکیک یا حساس سیاسی-اجتماعی جداً خودداری کنید.**
- این پروژه برای استفاده در بازی‌ها است، بنابراین لطفاً فقط کلماتی را اضافه کنید که در محیط‌های عمومی، آموزشی یا سرگرمی قابل استفاده باشند.
- کلمات باید از نظر املایی صحیح و متداول باشند.
- کلمات نباید شامل حروف و نشانه‌های پیچیده مانند «ء»، «أ»، «إ» و اعراب (َ ِ ُ ) باشند.
- از تکرار کلمات پرهیز کنید.

## فرمت فایل

- هر کلمه باید در یک خط جداگانه نوشته شود.
- فایل باید UTF-8 باشد.
- اگر ساختار فایل JSON است، به ترتیب حروف الفبا و بدون تکرار باشد.

## نحوه مشارکت

1. پروژه را Fork کنید.
2. تغییرات خود را اعمال کنید.
3. Pull Request ارسال کنید با توضیح کوتاه درباره‌ی کلمات اضافه‌شده یا ویرایش‌ها.

متشکریم که به بهتر شدن این منبع عمومی کمک می‌کنید!
